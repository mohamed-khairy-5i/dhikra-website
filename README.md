# ذِكْرَى · Dhikra

> حصنك اليومي من الأذكار والأدعية الصحيحة — مجاني، بدون إعلانات، ويعمل بدون إنترنت.
> Your daily fortress of authentic adhkar & duas — free, ad‑free, offline‑first.

🌐 **الموقع:** https://dhikra.netlify.app

---

## ✨ المميّزات (Features)

- **١٥ فئة أذكار** أصيلة من القرآن والسنّة مع المصدر والفضل لكل ذِكر.
- **عدّاد تفاعلي** لكل ذِكر مع تتبّع تقدّم وحفظ تلقائي.
- **مسبحة رقمية** بأهداف (٣٣ / ٩٩ / ١٠٠ / ∞)، إحصائيات، واهتزاز لمسي.
- **مساعد ذِكْرَى الذكي** — بحث دلالي عن الذِّكر المناسب حسب وصف الموقف، يعمل داخل الجهاز بالكامل (بدون إنترنت، بدون تتبّع).
- **أدوات المسلم** — اتجاه القبلة، مواقيت الصلاة، والتاريخ الهجري.
- **مدونة** بمقالات مؤصّلة عن الذِّكر والطمأنينة.
- **المفضلة** لحفظ أذكارك القريبة إلى قلبك.
- **ثنائي اللغة** (عربي/إنجليزي) مع دعم RTL/LTR ووضع ليلي.
- **PWA** يعمل بدون إنترنت ويُثبَّت على الجهاز.

## 🛠️ التقنيات (Tech)

HTML/CSS/JS ثابت (بلا إطار عمل) · نظام تصميم بـ CSS Custom Properties · Font Awesome 6.5.2 · خطوط Amiri / Tajawal / Sora · Service Worker + Web App Manifest · Schema.org.

## 📁 صيانة المحتوى (Maintenance)

```bash
python3 build_adhkar.py     # إعادة بناء صفحات الأذكار الـ15
python3 build_articles.py   # إعادة بناء مقالات المدونة
python3 build_index.py      # إعادة بناء فهرس مساعد ذِكْرَى
python3 inject.py page.html # حقن الهيدر/الفوتر في صفحة بها <!--NAV--> / <!--FOOTER-->
```
> بعد تعديل الأصول، ارفع رقم الكاش في `sw.js` لتحديث النسخة لدى المستخدمين.

راجع **[REPORT.md](REPORT.md)** لتقرير التطوير الكامل والخطة المستقبلية.
