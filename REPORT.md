## 2) الركائز الخمس (The 5 Pillars)

### الركيزة 1 — نظام تصميم فخم (Design System)
- لوحة ألوان مستوحاة من المخطوطات الإسلامية: **الزمرّدي / الذهبي / العاجي** (لا ألوان افتراضية باهتة).
- خطوط: **أميري** (عناوين عربية)، **تجوّل** (نصوص)، **Sora** (إنجليزي).
- Design tokens عبر CSS Custom Properties + وضع ليلي دافئ (midnight وليس أسود صرف).
- **العنصر المميّز (Signature element):** المسبحة الرقمية التفاعلية بحلقة تقدّم SVG ذهبية.

### الركيزة 2 — الحركة (Motion)
- ظهور تدريجي بالتمرير عبر `IntersectionObserver`.
- منحنيات نابضة فيزيائية `cubic-bezier(0.34,1.56,0.64,1)`.
- اهتزاز لمسي ذكي (`navigator.vibrate`) عند التسبيح.
- **احترام كامل لـ `prefers-reduced-motion`** (تعطيل كل الحركات لمن يفضّل ذلك).

### الركيزة 3 — الأداء (Performance)
- موقع ثابت خفيف بلا إطار عمل ثقيل — لا JavaScript زائد.
- صور محسّنة **WebP** مع بدائل JPG.
- **PWA offline‑first**: service worker (network‑first للصفحات، cache‑first للأصول).
- خطوط بـ `preconnect` + تحميل كسول للصور `loading="lazy"`.

### الركيزة 4 — تميّز الذكاء الاصطناعي (AI Differentiation)
- **مساعد ذِكْرَى** — بحث دلالي ذكي عن الذِّكر المناسب حسب وصف الموقف.
- **Scoped:** محصور في 108 ذِكر مفهرسة فقط.
- **Cost‑capped:** يعمل بالكامل داخل جهاز المستخدم — **صفر تكلفة API**.
- **Private & offline:** لا إرسال بيانات، لا تتبّع، يعمل بدون إنترنت.
- خرائط نيّة (Intent mapping): "ماذا أقول عند الخوف؟" → أدعية الكرب.

### الركيزة 5 — السرد ودعوة الفعل (Storytelling & Conversion)
- قوس سردي في الصفحة الرئيسية: آية → لماذا ذِكْرَى → العنصر المميّز → الفئات → دعاء اليوم → المدونة.
- **دعوة فعل واحدة واضحة (CTA) لكل قسم/صفحة**.
- مدونة حقيقية بثلاث مقالات مؤصّلة بدل "قريباً".

## 7) توسعة «ذِكْرَى 3.0» — استراتيجية منظّمة (مطبّقة ✅)

نُفّذت على أربعة محاور:

### المحور الأول — البنية التقنية والنشر
- `netlify.toml` + `_headers`: رؤوس أمان (X-Frame-Options، X-Content-Type-Options، Referrer-Policy، Permissions-Policy، HSTS) وتخزين مؤقت ذكي للأصول.
- `_redirects`: روابط نظيفة (`/asma`, `/names`, `/duas` …) بتحويلات 301.
- `404.html` صفحة خطأ مخصّصة أنيقة (noindex) + `offline.html` صفحة عمل بدون إنترنت.
- خدمة العامل ترجع الآن إلى `offline.html` عند انقطاع الشبكة.

### المحور الثاني — توسعة المحتوى
- **`asma-allah.html`** — أسماء الله الحسنى الـ99 كاملة مع معانيها، بحث فوري، مخطط بيانات ItemList (100) + BreadcrumbList.
- **`duas-quran.html`** — 20 دعاءً قرآنيًا مباركًا بالعربية والإنجليزية مع مصادرها.
- **`duas-nabawiyah.html`** — 20 دعاءً نبويًا صحيحًا مع تخريجها.

### المحور الثالث — تقوية SEO والفهرسة
- BreadcrumbList لكل الصفحات الجديدة ومقالات المدوّنة (Article + Breadcrumb).
- `sitemap.xml` = 31 رابطًا، تحديث `SiteNavigationElement` وفهرس الصفحة الرئيسية (11 قسمًا).
- ربط الصفحات الجديدة في الفوتر والتنقّل عبر 31 صفحة.

### المحور الرابع — أحدث تقنيات الويب (بترقية تدريجية آمنة)
ملف جديد `assets/js/modern.js` + طبقة CSS حديثة، كلها محميّة بكشف الميزات:
- **View Transitions API** — انتقالات صفحات ناعمة أصلية.
- **Speculation Rules API** — تحميل مُسبَق فوري للصفحات عند التمرير/التحويم.
- **CSS `:has()`، Container Queries، `@property`، `text-wrap: balance`، `color-mix()`، scroll-driven animations**.
- وعي بحالة الشبكة (offline/online) عبر التنبيهات.

**الاختبار:** صفر أخطاء كونسول عبر 33 صفحة (Playwright)، كل مخططات JSON-LD صالحة، الوضع الليلي سليم.

## 8) توقيع المطوّر + إصلاحات تدقيق شاملة (مطبّقة ✅)

### توقيع المطوّر — مكوّن واحد قابل لإعادة الاستخدام
- **`assets/js/credit.js`** — المصدر الوحيد للتوقيع. مفتاح إيقاف بسطر واحد (`SHOW_CREDIT = false`) يزيل التوقيع من كل الموقع دفعة واحدة.
- سطر فوتر صغير أنيق ثنائي اللغة (يبدّل تلقائيًا حسب لغة الصفحة): «© {السنة} · تصميم وتطوير محمد خيري · مهندس MERN Stack والذكاء الاصطناعي».
- **السنة تُشتقّ تلقائيًا** عبر `new Date().getFullYear()` — لا سنة ثابتة في أي مكان.
- أربعة روابط تواصل أيقونية: البريد (mailto)، الموقع الشخصي، GitHub، LinkedIn.
- مخطط بيانات **Person JSON-LD** (author/creator) يُحقن تلقائيًا في كل صفحة محتوى (name، alternateName، jobTitle، email، url، sameAs).
- مُطبَّق على 31 صفحة محتوى؛ صفحات المرافق الدنيا (404/offline) وملفات تحقّق Google مُستثناة عمدًا.

### إصلاحات تدقيق متعدّد الأدوار
1. إزالة «© 2025» الثابتة من 31 صفحة → السنة ديناميكية.
2. توليد `assets/img/og-image.png` (1200×630) لإصلاح معاينة المشاركة الاجتماعية المكسورة.
